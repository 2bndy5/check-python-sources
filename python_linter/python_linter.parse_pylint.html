    python_linter.parse_pylint.annotate_pylint_note(obj: dict) → str[source]¶

    Translate a 1 notification from pylint to github’s checks API.

    Parameters
    obj : dict

    The JSON object output by pylint (for 1 notification). A typical JSON object output from pylint looks like:

    {
    "type": "error",
    "module": "basic_test",
    "obj": "",
    "line": 3,
    "column": 19,
    "path": "tests/basic_test.py",
    "symbol": "syntax-error",
    "message": "invalid syntax (<unknown>, line 3)",
    "message-id": "E0001"
}

    Returns

    A str that can be used by github’s workflow log commands.
